WordPress plugin to add instant security basics

== Description ==

> This will automatically add basic security headers to any wordpress website.

= Overview =
Adding security layers to your website has never been easier. Just add the plugin and activate. With this plugin you will reach a score of A+ on https://securityheaders.com/

= Installation Instructions =
1. Upload `s2 Safety` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
